
# Alcance um Irmão — Plataforma Web (Versão Final)

Bem-vindo ao repositório da plataforma **Alcance um Irmão**, um protótipo funcional e visualmente completo de um portal para uma ONG fictícia que conecta pessoas em vulnerabilidade a profissionais voluntários.

## Conteúdo do pacote
- Páginas: index.html, about.html, how.html, projects.html, volunteer.html, help.html, contact.html
- Estilos: styles.css
- Scripts: scripts.js (form handling, LocalStorage, animações simples)
- Assets: assets/logo.svg, assets/images/banner-hero.svg
- README.md (este arquivo)

## Como executar localmente
1. Extraia o ZIP em uma pasta.
2. Abra `index.html` em um navegador moderno (Chrome, Firefox, Edge) ou use Live Server no VS Code.
3. Teste os formulários — os dados são salvos no `localStorage` para demonstração.
4. Para uso em produção, integre um backend para persistir dados e autenticação.

## Observações técnicas
- Site responsivo e mobile-first (Flexbox/Grid).
- Uso de SVG para logo e banners garantindo qualidade em qualquer resolução.
- Design com paleta acolhedora: azul suave e verde esperança.
- Scripts leves, sem dependências externas, fáceis de conectar a qualquer backend.

## Próximos passos recomendados
- Conectar a um backend (Firebase, Supabase ou API REST).
- Adicionar autenticação e painel administrador.
- Implementar envio de e-mails para confirmações/alertas.
